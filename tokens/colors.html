Accessibility

All color combinations meet WCAG 2.1 AA standards for contrast ratios:

  • Normal text: Minimum 4.5:1 contrast ratio
  • Large text: Minimum 3:1 contrast ratio
  • UI components: Minimum 3:1 contrast ratio
Color Pair Contrast Ratio Status
Primary (#0066cc) on White 4.78:1 AA Pass
Success (#28a745) on White 3.42:1 AA Large Text
Danger (#dc3545) on White 4.51:1 AA Pass
Gray 900 (#212529) on White 16.12:1 AAA Pass

Usage in Code

/* CSS */
.button-primary {
  background-color: var(--primary);
  color: var(--white);
}

.button-primary:hover {
  background-color: var(--primary-hover);
}

/* React/TypeScript */
import { colors } from '@albi/flight-tokens';

const Button = styled.button`
  background: ${colors.primary};
  color: ${colors.white};
`;
