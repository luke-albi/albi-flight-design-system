Usage in Code

All colors are available as CSS custom properties. Here's how to use them:

/* CSS - Using design tokens */
.button-primary {
  background-color: var(--albi-color-primitive-colors-core-blue-600);
  color: var(--albi-color-primitive-colors-core-neutral-white);
}

.button-primary:hover {
  background-color: var(--albi-color-primitive-colors-core-blue-700);
}

.button-danger {
  background-color: var(--albi-color-primitive-colors-core-red-600);
  color: var(--albi-color-primitive-colors-core-neutral-white);
}

.text-muted {
  color: var(--albi-color-primitive-colors-core-grey-500);
}

Accessibility

All color combinations are tested for WCAG 2.1 AA compliance:

  • Normal text: Minimum 4.5:1 contrast ratio
  • Large text: Minimum 3:1 contrast ratio
  • UI components: Minimum 3:1 contrast ratio for boundaries
Color Pair Contrast Ratio Status
Blue 600 (#1f51bc) on White 5.38:1 AA Pass
