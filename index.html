Welcome to Albi Flight

A comprehensive design system providing consistent, accessible, and beautiful UI components for web and mobile platforms.

Getting Started

Albi Flight is a design system that bridges design and development. Our documentation includes:

  • Component Documentation: Detailed specs, usage guidelines, and interactive examples
  • Design Tokens: Colors, spacing, typography, and more
  • Figma Integration: Direct links to Figma components with live screenshots
  • Platform Support: Components for web, mobile, and cross-platform use

Design Tokens

Design tokens are the visual design atoms of the design system. They define core values like colors, spacing, and typography.

Colors

Primary, secondary, semantic colors, and full color palette with accessibility considerations.

Spacing

Consistent spacing scale for margins, padding, and layout composition.

Typography

Font families, sizes, weights, and line heights for consistent text styling.

Border Radius

Standardized border radius values for component shapes and corners.
